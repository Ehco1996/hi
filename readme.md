# hi

hi is a tiny program that send benchmark request to "almost" everything. note that it's inspired by [rakyll/hey](https://github.com/rakyll/hey) and implemented by python/asyncio

## Install

 `pip install hi`

## Usage

* say hi to github

 `hi https://github.com`

* you will see
